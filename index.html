Welcome to API$WEB

API$WEB is a small collection of web projects created using real APIs. The main purpose of this project is to understand how APIs work, how data is fetched from servers, and how it is shown on a web page using JavaScript.

🎬 Movie Search Application

This project allows users to search for movies and view the top 20 movies. Movie data such as title, poster, release year, and ratings are fetched from a movie API. This project helps in understanding API fetching, search functionality, and dynamic content rendering.

🌐 Text Translator

The text translator project allows users to convert text from one language to another. It uses a translation API to process the input text and display the translated output. This project focuses on form handling, API requests, and real-time data processing.

⛅ Weather Application

This weather app shows real-time weather information of any city. Users can search for a city and get details like temperature, weather condition, humidity, and wind speed. This project demonstrates how live data is fetched and updated on a web page.
